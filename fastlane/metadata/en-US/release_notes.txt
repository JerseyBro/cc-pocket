What's new in 1.3.1

New
• Line-level diffs: changed files show +added / -removed counts, and each opens in a Diff | File viewer with collapsible hunks — on the phone, and in a new two-pane Changes browser on the desktop
• One row per running session: a project hosting several live sessions shows each one, and tapping resumes with the right backend (Claude or Codex)
• Default model for new sessions: pick it once in Settings
• Your computer shows up by its real name instead of an ID hash — a nickname you set still wins

Fixes & improvements
• Desktop: drag the sidebar divider to resize it, snap it hidden, bring it back from the edge strip; the close button on recent session rows actually works now
• Lists refresh the moment you come back, and the "running" badge turns off exactly when the turn ends
• Slash-command autocomplete leaves the cursor ready for arguments

Note: line-level diffs need the computer-side cc-pocket-daemon on 1.3.1 — run "cc-pocket-daemon update" (or brew upgrade / scoop update).
